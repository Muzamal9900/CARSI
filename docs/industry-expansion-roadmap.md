## Tier 1 — High Priority Industries (Create Pages)

### 1. Hospitality & Tourism

**Why:** Hotels, resorts, casinos, and cruise ships face frequent water damage, carpet wear, and require rapid restoration to maintain revenue.

| Discipline | Application                                      |
| ---------- | ------------------------------------------------ |
| WRT        | Flood response in guest rooms, basement flooding |
| CRT        | High-traffic carpet maintenance, stain removal   |
| ASD        | Pool/spa overflow, HVAC leaks                    |
| OCT        | Guest complaints about odours, smoke rooms       |

**Market Size:** 10,000+ hotels in Australia, $50B tourism industry

---

### 2. Education (Schools & Universities)

**Why:** Schools face mould issues (especially in older buildings), water ingress, and require safe environments for children.

| Discipline | Application                               |
| ---------- | ----------------------------------------- |
| AMRT       | Mould in classrooms, library water damage |
| WRT        | Roof leaks, bathroom floods               |
| CRT        | High-traffic corridors, assembly halls    |
| ASD        | Heritage school buildings                 |

**Compliance:** Education departments have duty of care requirements. IICRC credentials demonstrate competency for parent councils.

---

### 3. Insurance Industry

**Why:** Adjusters, assessors, and loss adjusters need to understand restoration scope to approve accurate claims.

| Discipline | Application                                |
| ---------- | ------------------------------------------ |
| WRT        | Understanding water categories and classes |
| FSRT       | Fire damage assessment for claims          |
| AMRT       | Mould remediation cost estimation          |
| ASD        | Structural drying scope of work            |

**Value Prop:** Insurers who understand IICRC standards can reduce over-scoping and fraudulent claims. Training for:

- Loss adjusters
- Claims assessors
- Building consultants
- Forensic accountants

---

### 4. Strata & Body Corporate

**Why:** Common property maintenance for 2.5M+ strata units in Australia. Building managers need credentials for compliance.

| Discipline | Application                                  |
| ---------- | -------------------------------------------- |
| WRT        | Basement flooding, roof membrane failure     |
| CRT        | Common area carpets, lift lobbies            |
| AMRT       | Mould in basements, subterranean carparks    |
| ASD        | Post-flood structural drying in multi-storey |

**Compliance:** Strata Schemes Management Act requires proper maintenance. IICRC credentials protect building managers from personal liability.

---

### 5. Retail & Shopping Centres

**Why:** Major landlords (Scentre, Vicinity, GPT) require certified contractors for tenant restoration work.

| Discipline | Application                      |
| ---------- | -------------------------------- |
| WRT        | Sprinkler activation, roof leaks |
| CRT        | Tenant flooring restoration      |
| OCT        | Food court odour control         |
| FSRT       | Fire damage in retail precinct   |

**Value Prop:** Pre-qualification for major landlord contractor panels.

## Government Contractor Value Proposition

### AusTender Pre-Qualification

IICRC certification as a **mandatory requirement** for:

- Commonwealth facility maintenance contracts
- Defence base remediation
- Heritage building restoration
- Emergency response panels

### State Government Panels

| State | Panel                       | IICRC Requirement       |
| ----- | --------------------------- | ----------------------- |
| NSW   | Construct NSW               | Recommended             |
| VIC   | Building Remediation Panel  | Required for Category C |
| QLD   | QBuild                      | Recommended             |
| WA    | Building Management & Works | Required for heritage   |
| SA    | DPTI                        | Recommended             |

### Local Council Contracts

537 councils across Australia require:

- Building maintenance contractors
- Emergency response providers
- Heritage conservation specialists
- Stormwater damage response

### Defence Base Maintenance

- Base maintenance contracts require certified personnel
- Heritage buildings on defence land (e.g., Victoria Barracks)
- Rapid response for classified facilities

## Add-On Certifications for Cleaners

Position IICRC certifications as **competitive differentiators** for commercial cleaners:

### The "Cleaning Plus" Value Proposition

| Base Service        | Add-On Certification | Upsell Value                 |
| ------------------- | -------------------- | ---------------------------- |
| General cleaning    | + WRT                | Offer flood response         |
| Carpet cleaning     | + CRT                | Insurance restoration work   |
| Office cleaning     | + AMRT               | Mould inspection/remediation |
| Commercial cleaning | + OCT                | Odour complaints resolution  |
| Aged care cleaning  | + AMRT + CRT         | Infection control compliance |

### Premium Contractor Positioning

**"Cleaning contractors who hold IICRC certifications can charge 30-50% more by offering restoration services, not just maintenance."**

Key messaging:

1. **Differentiate** from "mop and bucket" competitors
2. **Diversify** revenue with insurance restoration work
3. **Qualify** for government and corporate tender panels
4. **Protect** with documented competency (WHS defence)
