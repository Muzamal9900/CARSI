# Industry Association Partnerships — CARSI LMS

## GP-148 | Priority: P1 High

## Objective

Establish CARSI as a recognised training partner across 7+ Australian industry associations. Convert association endorsements into group licensing revenue and a steady pipeline of member enrolments.

**Targets:**

- 3 signed MOUs within 6 months
- 5 active partnership agreements within 12 months
- 200+ enrolments sourced via association referral codes per year

## Partnership Tiers

### Tier 1: Endorsed Provider (Entry Level)

| Benefit                 | Detail                                                                                                     |
| ----------------------- | ---------------------------------------------------------------------------------------------------------- |
| **For the Association** | Free listing in training directory, co-branded completion certificates for members, no cost to association |
| **For CARSI**           | Logo on CARSI website ("Endorsed by..."), access to member communications, credibility signal              |
| **Member Benefit**      | 10% discount on all CARSI courses via unique member code                                                   |
| **Revenue Share**       | None                                                                                                       |
| **Commitment**          | 12-month trial, renewable annually                                                                         |
| **Cost to Association** | $0                                                                                                         |

### Tier 2: Preferred Training Partner (Mid Tier)

| Benefit                 | Detail                                                                                                                   |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------ |

| **Member Benefit**      | 15% discount on all CARSI courses, priority access to new courses, free lead magnet PDF                                  |
| **Revenue Share**       | None (discount funded by CARSI marketing budget)                                                                         |
| **Commitment**          | 12-month agreement, auto-renew                                                                                           |
| **Cost to Association** | $0                                                                                                                       |

### Tier 3: Exclusive Training Partner (Top Tier)

| Benefit                 | Detail                                                                                                                                                               |
