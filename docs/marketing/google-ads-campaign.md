## Bidding Strategy

| Parameter                  | Setting                                                     |
| -------------------------- | ----------------------------------------------------------- |
| Strategy                   | Target CPA (tCPA)                                           |
| Target CPA                 | $55 AUD (midpoint of $45–65 range)                          |
| Initial phase (weeks 1–4)  | Maximise Conversions (let Google learn)                     |
| Transition to tCPA         | After 30+ conversions accumulated                           |
| Bid adjustments — Device   | +15% mobile (tradies search on phone)                       |
| Bid adjustments — Location | +20% WA, QLD (mining); +10% NSW, VIC (restoration/cleaning) |
| Bid adjustments — Schedule | +10% Mon–Fri 6am–9am, 7pm–11pm (before/after shift)         |

---

## Budget Recommendation

| Phase                    | Monthly Budget | Focus                                                         |
| ------------------------ | -------------- | ------------------------------------------------------------- |
| Month 1–3 (Learning)     | $3,000 AUD     | All 6 campaigns, even split, gather data                      |
| Month 4–6 (Optimisation) | $4,000 AUD     | Shift budget to top 3 performing verticals                    |
| Month 7–12 (Scale)       | $5,000+ AUD    | Double down on ROAS-positive verticals, pause underperformers |

**Break-even calculation:** At $55 CPA and $295 average course value, ROAS = 5.4x. At $795 subscription, ROAS = 14.5x.
